## Deployment

To deploy this project run first clone it !

```bash
  git clone https://github.com/miladbasery/py-calculator.git
```
open file
```bash
  cd py-calculator
```
install python
```bash
  pip install py
```
install library
```bash
  pip install tkinter
```
